# Plane-Alert candidate filters
#
# Format:
#   ICAO:<glob-pattern>:<owner fallback>
#   CALLSIGN:<glob-pattern>:<owner fallback>
#
# Matching logic is OR:
#   a record is considered if ICAO matches any ICAO pattern OR callsign matches any CALLSIGN pattern.
#
# Notes:
# - Matching is case-insensitive (internally uppercased)
# - Patterns use shell-style globs (* and ?)
#
